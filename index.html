What is jQuery?

Before you start studying jQuery, you should have a some basic knowledge of:

  • HTML
  • CSS
  • JavaScript

jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code. jQuery also simplifies a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.

Why?

One of the best uses of jQuery is the ability to modify pages and animate with little effort.
But the best use of jQuery is using plugins to acomplish advanced features such as: slideshows, touch swipe, paging, and more

How to Use

In order to use jQuery, you first need to link to a library in your header (using Google is a good option).

<head> <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script> </head>

Once you have linked the main jQuery library, you can now begin using jQuery coding or linking in plugins from other websites to create galleries and slideshows.

Coding

Once you have linked the jQuery library in your head you can begin using jQuery specific code.

You always code inside the "<script>" tags. and the first line of your code will start with:

  $(document).ready(function(){

and will end with:   });

A basic code setup looks like this:

<script> $(document).ready(function(){ //Code here }); </script>

For more information on commands and jQuery specific coding, go to there website: jQuery Coding or w3Schools or view the examples for jQuery below.

Using Plugins

To use a jQuery plugin is fairly straight forward. You must first download and link the plugin in your header (or link directly from the developer's website). The developers of these plugins always have documentation and direct examples on how to setup and use their plugin (be sure to read carefully).

<head> <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script> <script src="js/plugin.js"></script> </head>

After you have linked to the plugin (that you either downloaded to your "js" folder or directly from their page), you just need to follow the code provided on the website or on the github repository. Developers provide you with the full html layout code and jQuery code (plus CSS code if it is needed) needed to use their plugin.

What is CSS3?

“CSS3” simply refers to the latest incarnation of CSS, with additional capabilities far beyond the scope of the first two generations.

What Does It Do?

CSS3 allows developers to build content-rich web pages with relatively lightweight code requirements. A website that is not dependant on parsing JavaScript code or relying on Flash animations can load at much quicker speeds and still have the same effects and in some cases better. jQuery in some instances even relies on the animations and transitions of CSS3 to execute its design.
Using CSS3 allows designers to be able to create multi-column layouts, border effects, gradients, shadows, and more. Ther is no special pieces of code to include, there isn't anything special that has to be downloaded; it's all just based on simple browser supported functions.
CSS3 has many new modules that build on the old ones, and some that are completely new. Some of the most important newer modules are:

  • Selectors
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout
  • User Interface

Warnings

All modern browsers support most parts of CSS3, but older browsers like, IE8 and below, do not. The more advanced uses might not work on some newer browsers even, use it carefully.

Using CSS3 over jQuery can create faster load times of a website and cut down on the website becoming slow. CSS3 or jQuery both put stress on a users browser and computer and both can slow or stop the computer.

Using a simple button to add and remove class with "toggle"

The HTML and CSS mark-up is a regular <button> and a <div> with 2 defined CSS classes:

HTML

<input id="MyButton" type="button" value="Click me!" onclick="Toggle();" /> <div id="MyDiv" class="RedClass" >Hello World!</div>

CSS


.RedClass { background-color: Red;} .BlueClass{ background-color: Blue; }

The only thing that has been added is in the button, a new line of code:

onclick="Toggle();"

"onclick" is a jQuery command that activates a function when the element is clicked (in this case it is the button, but it can be anything). The function "Toggle()" is defined inside the "<script>" tags in our page. All jQuery script needs to be preceded by a $.

<script> function Toggle(){ $("#MyDiv").toggleClass("BlueClass"); } </script>

"Toggle()" use is to use the jQuery command ".toggleClass" and is selecting the any element that has the id "MyDiv". It does exactly what it says it does, it toggles a class on and off. When the button is clicked all elements with the id "myDiv" gain a 2nd class (in our case it is only our <div>):

<div id="MyDiv" class="RedClass BlueClass">Hello World!</div>

Since BlueClass is lower on our style sheet and is being activated by our jQuery, it take priority and changes the background. A second class removes the class.

That is a how to toggle a class on and off with a button

Hiding/Showing an Element

A great technique used with jquery is the ability to hide and/or show any element (img, text, div, etc) with any given criteria. You can have a box appear after a user scrolls so far down or if they click a button. The jQuery code is .hide() to hide something and .show() to show things; you can also use the .toggle() command to turn on an off like a switch.

Lets create a a box that appears and hides on the click of a button.

The first thing we need to do is create a button and a then a div to play hide and seek with. This time we'll make it an HTML5 button, because that is the best practice.

HTML

<button>Click me!</button> <div id="mydiv">This is Only a Test</div>

Now for some CSS to make the div look pretty and make a letterpress text effect.

CSS

#mydiv { height:200px; width:400px; background: #323232; color:#222; text-shadow: 0px 1px 2px #555; display:none; text-align:center; line-height:4em; font-size:3em; }

Now we have our CSS and HTML and our div is hidden. No for the jQuery. Unlike before we didn't attached an onclick function to our button, doing this means all we have to do is declare a function and what that function does. So we're going to tell the browser that when the HTML5 button is pressed $('button').click(function () { that it needs to preform a function, and that function is to .toggle() the visibility of the div.

jQuery

$('button').click(function () { $('#mydiv').toggle() });

Now that we have that setup, you could easily switch .toggle() out for .slideToggle() or .fadeToggle() to make it animated. Each of these functions also have functions that can specify time it takes to appear and more.

Make an Element Appear at a Certain Scroll Point

So something that is always aesthetically pleasing is to create an event that is triggered by scrolling. Now there are may methods to go about doing this, but this is a quick, simple, and easy way. For this we need to use the $(window).scroll(function() { command. So for this we're going to have button fade in if the user has scrolled more than 300px from the top of the page; and then we're going to also say if the user scrolls more than 1000px from the top it will fade back out. This is also a bit tricky, the key to the pixels is the offset from the scroll, meaning it isn't 100% exact like a div; though you can fix that problem too with more coding.

The first thing we need is to setup our button and make our page long enough.

HTML

<div id="mydiv"> <button>Click Me!</button> <p> This is Only a Test </p> </div>

CSS

body { height: 2000px; } button { position:fixed; top:20px; left:30px; display:none; } #mydiv { background:red; height:1000px; width:500px; }

So now we have a small setup to see what's going on, we can throw in a bit of jQuery. The first thing is setting up the variable for the Y-axis scrolling, so the browser knows to watch the distance of the scroll. var y_axis = window.pageYOffset; using that code the browser will look for the scroll distance and store the information. All that's left to do is provide and if/else statement with the distances we want, followed by the function to preform.

jQuery

$(window).scroll(function() { var y_axis = window.pageYOffset; if(y_axis > 300 && y_axis < 1000) { $('button').fadeIn(); } else { $('button').fadeOut(); } });

And that is all there is. No when the scroll point gets past 300px the button will fade in, but if it scrolls more than 1000px it will fade back out.

Animating

The next big jQuery use is animating. With jQuery you can animate objects with very little code.
So lets animate a square div 85% of the container and have it take 1 second to get to the end and then take 2 seconds to come back to its original position.

Create a basic div with a button to activate the animation. (You can set the parameters to anything really, you can it based on a link or if a person scrolls so far down the page, etc...)

HTML

<button>Start Animation</button> <div id="mydiv"> Testing </div>

Now add the styles to the div.

CSS

#mydiv { background:#98bf21; height:4em; width:4em; position:relative !important; line-height:4em; font-size:1.5em; text-align:center; margin:15px 0; }

No we have a nice green square box with some centered text inside.

Note the position property has to be set!

Now lets add the script. We're going to use the .animate() command and give the information needed, .animate( properties, duration). The animate command has many other available options, go to the jQuery website for more information about the command. For our purposes we're going to just give it a left css property and a time to complete the trasition.

jQuery

$(document).ready(function(){ $("button").click(function(){ $('div').animate({left:'90%'}, 1000); $('div').animate({left:'0'}, 2000); }); });

Now it will go to one end and then back in a span of 3 seconds.

Pseudo Elements

One of the big advantages to come from CSS3 are the new pseudo elements. ":focus, :first-letter, :first-line, :first-child, :before, :after" they are named for their functions (you can find them all here).

The best used two are the ":after" and ":before" elements. With these we can create extra elements around our existing elements. These two are generally used for extra decorations or to create close buttons (like used in this div on the top right). The most common usage is to create speech bubbles, icons/extra buttons, 3D shadow effects, or ribbon navigation bars. You can see a speech bubble to the right.

Why don't we just create a simple icon outside of a nav element to give it some flare.

First we're going to need a simple navigation setup.

HTML

<ul> <li class="play"><a href="#non">Play</a></li> <li class="stop"><a href="#non">Stop</a></li> <li class="pause"><a href="#non">Pause</a></li> </ul>

SO first thing we have to so is setup our listed items. We want them off the wall and have a big enough height to match our icons. margin-left:20px; height:25px;
Next we set the parameters for all the pseudo elements. we know our icons are roughly 15x20 pixels and we need them off the wall too. We are using absolute positioning so they are easier to deal with and to move around; as well as if they aren't absolute positioned then we can't have blank spaces in the content property (the content property is the key element for pseudo classes).

li:before { content: " "; position:absolute; left:2px; background: #888 url('http://us.cdn2.123rf.com/168nwm/ionprodan/ionprodan1103/ionprodan110300042/9023096-set-of-player-icons-such-as-play-pause-and-other.jpg'); width: 15px; height: 20px; }

now that we have set our background and our positioning; plus the sizes of all our pseudo elements, all we need to do is move the background around with the background-position: property in the CSS.

when it's all together it looks like this:

CSS

li {margin-left:20px; height:25px;} li:before { content: " "; position:absolute; left:2px; background: #888 url('http://us.cdn2.123rf.com/168nwm/ionprodan/ionprodan1103/ionprodan110300042/9023096-set-of-player-icons-such-as-play-pause-and-other.jpg'); width: 15px; height: 20px; } .play:before { background-position: -30px 0; } .pause:before { background-position: -52px -2px; } .stop:before{ background-position: -104px 0; }

And that is it. No extra elements, or jQuery needed.

General and Adjacent Sibling Combiners && Child combinators

A Child Combinator describes a childhood relationship between two elements. You can select an element that is a descendant of an another element to make conditional CSS. The way it is denoted is by the usage of the "greater-than sign" (>). Say you have several elements nested with other elements. (

e.g. a bunch of <p> elements nested within a div, but in than div is more divs with other <p> elements. Instead of going through and adding classes to all the ones that direct children of the first div and adding a class, you can just use a sibling combiner to select those <p> that are direct children of the div.
See Example 1

HTML

<div id="one"> <p>Paragraph one</p> <div> <p>Paragraph two</p> </div> <p>Paragraph three</p> <div> <p>Paragraph four</p> </div> <p>Paragraph five</p> <div> <p>Paragraph six</p> </div> <p>Paragraph seven</p> <div> <p>Paragraph eigh</p> </div> </div> <div id="two"> <p>Paragraph one</p> <div> <p>Paragraph two</p> </div> <p>Paragraph three</p> <div> <p>Paragraph four</p> </div> <p>Paragraph five</p> <div> <p>Paragraph six</p> </div> <p>Paragraph seven</p> <div> <p>Paragraph eigh</p> </div> </div>

we want only the <p> that are direct children of the div; one, three, five, and seven. Using this a child combiner we can target them all with one line of code: #two > p { background:blue; }. This finds all those <p> elements and turns their background blue.

CSS

#one { height:300px; background:blue; width:25%; float:left;} p { color:white; background:#333; height:15%; text-align:center; line-height:2em;} div > div { height:15%; background:#333; } #two { height:300px; background:blue; width:25%; float:left; margin-left:20px;} #two > p { background:blue; }

General and Adjacent Sibling Combiners work in a similar fashion to Child Combiners. There are two different sibling combinators: the adjacent sibling combinator and the general sibling combinator. In both cases, non-element nodes (e.g. text between elements) are ignored when considering adjacency of elements.

The adjacent sibling combinator is made of the "plus sign" (+) character that separates two sequences of simple selectors. The elements represented by the two sequences share the same parent in the document tree and the element represented by the first sequence immediately precedes the element represented by the second one.

Broken down, this means that the an element will find the nearest sibling that matches the criteria and select it. p:first-of-type + div this will find the first <p> element and then find the closest div in the same level and target it for modification See Example 2

The general sibling combinator is made of the "tilde" (~) character that separates two sequences of simple selectors. The elements represented by the two sequences share the same parent in the document tree and the element represented by the first sequence precedes (not necessarily immediately) the element represented by the second one.

Meaning that #two p ~ div will find all of its immediate siblings and target them for alteration.

Transitions

CSS3 has made it possible to stop relying so much on jQuery (it can slow a web page down), we can now create transition effects and more all with CSS. Like the jQuery example of a slide out div, that can be created with some quick and easy CSS3.

Since we're not using jQuery, we have no need to have a button, instead we just need a check box input.

HTML

<input type="checkbox" name="chk" id="toggle-1" > <div id="slide"> <p> Some Text </p> </div>

And now to give it some style

CSS

#slide { display:block; height: 0px; width:300px; background:#333; color:#fff; } input#toggle-1:checked ~ #slide { position:relative; height: 200px; display:block; width:300px; } input, div {margin-left:25px;}

Now our checkbox switches on and off the height of the div. Now we just need to animate it with some transitions. We add them to #slide.

#slide { display:block; height: 0px; width:300px; background:#333; color:#fff; transition: all .7s ease-out; }

And that is all, that 1 line of code adds the animation ability to the #slide div. There are in some cases need of adding multiple versions of that code with "vendor prefixes" to allow it to animate on other browsers like Internet Explorer and Mozilla. You can use http://cssprefixer.appspot.com/ to help you fix your code when needed.

:nth-child Selector

This is by far an amazing asset to web development. Using the :nth-child selector you can now select the first child, all the even (or odd) children, even use mathematical equations to select ever 5/6/7th child, last child, and more. A great use for it is if you have a list or a bunch of elements listed and you want some seperation between them all, to show there are different sections or parts.

Assume you have a standard unorded list with a lot of list elements in it with a bunch of information or you're using it as a slide holder. To show a seperation between all the children, you would normally have to add a class to ever-other list element and then style. With the :nth-child(odd) and/or :nth-child(even) to sperate them in some fashion. Along with :nth-child there is also :last-child and :first-child. You can also use the similar pseudo selectors to select the first line or first letter of an element.
